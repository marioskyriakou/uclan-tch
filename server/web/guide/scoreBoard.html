API Guide: scoreBoard

Usage

Retrieves the current score board which includes all the players for the category of the specified session.

URL format:
http://uclan-thc.appspot.com/api/csv/scoreBoard?session=...

Parameters

  • session: the session ID, typically retrieved after calling startQuiz

Possible outcomes

  • When: Everything OK
    Status: "OK", Message: ""
    Second line and on: the score of individual sessions (i.e. the app name, the user name, the score, and the finish time-which can be 0 if not finished)
    Example: 
    OK,
coolApp2,MyUser,0,0
MyApp,Adam,10,0
coolApp,Nearchos,30,548541
  • When: One or more parameters are missing
    Status: "Invalid or missing parameters", Message: "Valid parameter list 'scoreBoard?session=...'"
    Example: 
        Invalid or missing parameters,Valid parameter list 'scoreBoard?session=...'
  • When: The session is unknown
    Status: "Unknown session", Message: "The specified session ID could not be found"
    Example: 
        Unknown session,The specified session ID could not be found

Example

Issuing this URL 
http://uclan-thc.appspot.com/api/csv/scoreBoard?\
                    session=agtzfnVjbGFuLXRoY3IUCxIHU2Vzc2lvbhiAgICAgMOVCgw

returns 
OK,
coolApp2,MyUser,0,0
MyApp,Adam,10,0
coolApp,Nearchos,30,548541
