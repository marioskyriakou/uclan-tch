API Guide: categories

Usage

Retrieves the list of available categories. Can be called at any time without a player name or session ID. Retrieves a list of categories.

URL format:
http://uclan-thc.appspot.com/api/csv/categories

Parameters

No parameters expected.

Possible outcomes

  • When: Everything OK
    Status: "OK", Message: ""
    Followed by lines of: "uuid,name"
    Example: 
                OK,
            agtzfnVjbGFuLXRoY3IVCxIIQ2F0ZWdvcnkYgICAgIC6hwoM,Treasure hunt 2013
            agtzfnVjbGFuLXRoY3IVCxIIQ2F0ZWdvcnkYgICAgICWvQoM,Treasure hunt 2014
  • When: No categories are found on the server
    Status: "Empty", Message: "No categories found"
    Example: 
                Empty,No categories found

Example

Issuing this URL 
            http://uclan-thc.appspot.com/api/csv/categories

returns 
            OK,
            agtzfnVjbGFuLXRoY3IVCxIIQ2F0ZWdvcnkYgICAgIC6hwoM,Practice Treasure Hunt Cyprus 2013
            agtzfnVjbGFuLXRoY3IVCxIIQ2F0ZWdvcnkYgICAgIC6hwkM,Testing
